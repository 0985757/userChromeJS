AddonsPage.uc.js
================

附件组件页面

 - 右键新增查看安装目录（支持扩展、主题、插件）、复制名字
 - 详细信息页面新增GM脚本、扩展、主题安装地址和插件路径，右键即复制。

根据 [Add InstallUrl Or Path To AddonsPage](http://j.mozest.com/zh-CN/ucscript/script/109/) By Crab 修改，参考 [Add-ons Manager Context Menu 扩展](https://addons.mozilla.org/zh-cn/firefox/addon/am-context/) 和 OpenAddonFolder.uc.js
